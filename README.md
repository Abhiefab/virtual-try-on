# 🕶️ Virtual Try-On Web App

A stylish web app that allows users to try on different glasses virtually by uploading an image and positioning glasses using drag, rotate, and resize tools.

## 🚀 Features

- Upload your own photo
- Choose from multiple glasses styles
- Auto face detection with face-api.js
- Manual edit mode (drag, rotate, resize)
- Download the final image
- Sleek, glassmorphism UI

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- [face-api.js](https://github.com/justadudewhohacks/face-api.js)

## 📷 Glasses Styles

- Rectangle
- Rectangle 2
- Retangle 1
- Round

## 🖼️ How to Use

1. Click **Upload Image** and select your photo.
2. Pick a glasses style from the buttons.
3. Click **Edit Manually** to fine-tune position and size.
4. Download your image with the button below.
